Sistemas:

Seleccione un sistema de aqui abajo para obtener ayuda: (no se a incluido Writing Machine por que demomento no tiene inovaciones)

Estos son los comandos de la ultima version:

  • exit -> Exit the system.
  • .help -> Shows this list.
  • get-fs -> Shows all files in the system folder
  • cat --ver -> Shows the version of the OS
  • print 'text' -> Prints the variable text with the determinated options
  • cat -sta-key -> Shows the start key
  • cat -debug 'filename' -> Runs the selected .cpm file
  • clear -> Clears the window
  • pause -> Pause the system
  • 'filename'.cpm or 'filename' -> Debugs a .cpm file
  • readf 'filename' -> Shows the file content
  • lbat-c 'command' -> Run the selected cmd command
  • in -> Strores a user input in a variable called '(in)'
  • date -> Changes the date
  • get-cpm -> Gets all .cpm files in the directorie
  • cal 'filename without .cal.d' -> Opens the selected file in calendar
  • notes 'filename without .no.t' -> Open notes with the selected file.
  • copy 'text' -> Stores the text in a variable called '(paste)'
  • calc 'calculate' -> Shows the operation result. It can be called to return the calc as '(calc='calculate')'
  • var 'numero' 'texto' -> Saves a variable in the number selected and with the text selected and it can be called with '(vars='number')'
  • snake.cpm -> Runs snake
  • setcopy.cpm -> Run the copy set
  • chamap -> Selects a character of ASCII and put its in the clipboard
  • lbat-c chapmapmap.exe -> Shows all characters ASCII code
  • printp 'message' -> Shows the selected text with no new line
  • inpa -> Makes a input with new line at the start

Aqui hay algunos ejemplos de archivos que puedes ejecutar poniendo "'nombre del archivo'.cpm":

Hello World!:

print Hello World!

La funcion print llama al copilador a poner el mensaje seleccionado

Calculadora:

printp Introduzca un calculo:
in
printp La solucion es:
calc (in)

La funcion printp hace que el texto no tenga una linea nueva y llama a una entrada de texto que luego se llama en la funcion "calc" y se hace le calculo. El sistema deberia dar un pantallazo azul despues de que el resultado no sea un numero.

Iniciar sesión:

print Iniciar sesion
printp Nombre:
in
var 0 (in)
printp Contraseña:
inpa
var 1 (in) print Gracias por iniciar sesion

La funcion var guarda los valores con un numero y se pueden llamar como "(vars=0)" = Usuario y "(vars=1)" = Contraseña. Luego, el comando inpa llama una entrda de texto pero tapa la entrada.
